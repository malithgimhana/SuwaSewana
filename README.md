# SuwaSewana - Doctor and Patient Management System

<img src="img/Github Description voer phot.png" alt="Logo" width="300" title="Cover photo">


SuwaSewana is a web-based healthcare management system with role-based dashboards for Admin, Doctor, and Patient users. It includes doctor/patient management, appointment scheduling, profile management, authentication, and secure password handling with backend validation. Built with PHP, JavaScript, HTML, and CSS for a complete end-to-end clinic workflow.
